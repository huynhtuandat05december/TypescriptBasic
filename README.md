# Typescript Basic
### **How to run ts file**

*First Way*

Complier ts file with command

```
tsc index.ts
```
it will complier to a js file, then
```
node index.js
```

*Second Way*

install ts-node (same node for js)

```
npm install -g ts-node
```

Then run ts file with command
```
ts-node index.ts
```
